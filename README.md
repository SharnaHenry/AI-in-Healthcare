AI-In-Healthcare

A multi-page educational website exploring the applications of Artificial Intelligence in modern healthcare.

# Overview
This project is a responsive website designed for technology and medical students who are interested in understanding how Artificial Intelligence is transforming healthcare.

The website explores topics such as:
- AI in diagnostics
- Machine learning in medical imaging
- Predictive analytics
- Ethical considerations of AI in medicine


# Technologies Used
- HTML5
- CSS3
- Responsive design principles
- Git & GitHub for version control

# Features
- Multi-page navigation structure
- Separate mobile-optimized layout
- Clean UI with custom CSS styling
- Structured educational content

# How to Run

1. Clone the repository
2. Open Index.html in your browser
